## 글자 태그

### 제목과 본문 글자 태그

- 제목 글자 태그에서 h1는 heading(제목)을 의미한다.
- p는 paragraph(단락), br은 break(줄 바꿈), hr은 horizontal rule(수평 줄)을 의미한다.

h1이 가장 큰 제목 태그이고, h6이 가장 작은 제목 태그이다.

```
<body>
    <h1>제목 글자 태그 1</h1>
    <h2>제목 글자 태그 2</h2>
    <h3>제목 글자 태그 3</h3>
    <h4>제목 글자 태그 4</h4>
    <h5>제목 글자 태그 5</h5>
    <h6>제목 글자 태그 6</h6>
</body>
```
